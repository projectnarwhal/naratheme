NarAtheme

NarAtheme is a fork of Atheme customized to add additional integration with projectNarwhal through the Site Integration Service (SiteServ).

Installing NarAtheme

Note: The following is not meant to be copy-pasted. Please read through and understand what each of the following commands does before running them.

$ export INSTALL_DIR=/srv/naratheme
$ git clone git://github.com/projectNarwhal/naratheme.git
$ cd naratheme
$ git submodule init
$ git submodule update
$ ./configure --enable-contrib --prefix=$INSTALL_DIR
$ make -j5 # Change to [CORES + 1]
$ sudo make install
$ cd $INSTALL_DIR
$ cp etc/atheme.conf.example etc/atheme.conf
$ vim etc/atheme.conf
$ ./bin/atheme-services
